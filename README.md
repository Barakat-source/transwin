# Web-App : Générateur d'Ordres de Mission (e-OM) 🚛

Projet de stage réalisé pour l'entreprise **TRANSWIN**.

## Objectif
Digitaliser la création des voyages et supprimer les saisies manuelles.

## Fonctionnalités
1.  **Base de données SQL** : Gestion des Clients, Camions, Chauffeurs et Voyages.
2.  **Formulaire de saisie** : Interface pour créer un nouveau voyage avec affectation chauffeur et camion.
3.  **Générateur PDF** : Création instantanée d'un document PDF professionnel.
4.  **Tableau de bord** : Historique des derniers voyages créés.

## Technologies
- PHP 8.2
- MySQL
- Tailwind CSS
- FPDF
